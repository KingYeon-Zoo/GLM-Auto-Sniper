# 🚀 GLM-Auto-Sniper (智谱 GLM 终极自动抢购助手)

> 一款基于油猴 (Tampermonkey) 平台的自动化脚本，专注于为智谱大模型平台提供高强度、自动化的安全抢购解决方案。拥有极客风赛博朋克控制面板，真正做到美学与性能兼顾。

## ✨ 核心特性 / Features

*   **🛡️ 双防线请求拦截**：在底层同时拦截 `Fetch` API 和 `XMLHttpRequest`，精准捕获原始抢购请求，跳过繁琐的前端动画与限制。
*   **🔁 高速引擎与双重校验**：不仅全自动重试 `/preview` 接口，还会调用 `/check` 接口对凭证有效性进行双重严苛校验，确保真正抢购成功，避免“假成功”。
*   **🤖 弹窗自动恢复机制**：智能监听前端抛出的“系统繁忙”、“请重试”等各类错误弹窗（支持多种常见组件库如 Element UI, Ant Design 等），全自动隐藏弹窗并重新触发拦截流。
*   **🎛️ 极客前卫控制台 (Tactical HUD)**：摆脱粗糙的默认界面，内置赛博前卫 (Cyber/Brutalist Neon) 风格控制浮窗。
    *   动态呼吸灯指示器 / 实时请求分析展示 / HUD 终端日志轮播面板。
    *   全玻璃毛化 (Backdrop Blur) 设计，同时支持自由全局拖拽。

## 📦 安装与使用 / Installation & Usage

1. **安装环境**：请确保您的浏览器已安装 [Tampermonkey (油猴插件)](https://www.tampermonkey.net/)。
2. **导入脚本**：
    *   在 Tampermonkey 控制面板中点击 `添加新脚本`。
    *   将本项目目录下的 [`GLM自动抢购.js`](GLM自动抢购.js) 文件内容全部复制并粘贴到编辑器中。
    *   使用 `Ctrl + S` 或 `Cmd + S` 保存。
3. **开始使用**：
    *   打开智谱模型相关网站。
    *   屏幕右上角将呼出 `GLM自动抢购` 控制台 (System Standby 待命状态)。
    *   您可以根据需要调节 **重试间隔**（如缩短至 50ms）或 **上限次数**。
    *   **正常在全网发起一次点击购买** —— 脚本会自动劫持您的点击，接管控制权，随后疯狂以设定极速运作，直至捕获到有效商品。

## ⚙️ 界面演示

由于这是一款注入式脚本，其拥有独立运行的生命周期。
面板状态说明：
*   `SYSTEM STANDBY` (待机状态)：此时您需要先点击页面上的“购买/抢购”按钮来让脚本捕获原始请求特征。
*   `EXECUTING` (黄色高亮呼吸)：系统正在毫秒级连发突围！此阶段可一键强行中止操作。

*   `FAILED` (红色警告)：达到重试阈值，自动停止运行。
